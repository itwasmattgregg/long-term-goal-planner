# Horizon

A local-first PWA for long-term goals. Write plain-text sentences; the app links them to existing goals or creates new ones, tracks wins and misses, and surfaces a small daily focus so multi-year goals feel like leveling up.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Data stays in your browser (IndexedDB). No account required.
